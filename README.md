# FastAPI Sandbox

FasstAPIのサンプルコード集

## Prerequisite

- pyenv
- uv

## Commands

### Library

```
$ uv sync
```

```
$ uv add library
```

```
$ uv remove library
```

### Run scripts

```
$ fastapi dev
```

```
$ uv run script
```
